FMSec (File Manager SECurity) - a set of security extensions to file managers.

To install:
- change into the directory of your chosen desktop environments
  (e.g. 'cd KDE3'),
- run './install.sh'

To uninstall, if needed:
- change into the directory of your chosen desktop environments
  (e.g. 'cd KDE3'),
- run './uninstall.sh'
